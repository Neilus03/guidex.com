Abstract

IE systems often fail when label definitions change. GUIDEX fixes that: it infers a domain schema, writes guidelines, and generates rule-consistent examples. Fine-tuning LLaMa-3 8B on those examples lifts zero-shot NER by up to +7 F1 on seven benchmarks and still helps when gold data is present. The paper, model and data will be released.

Methodology  4 steps

  1. Summarise. The LLM turns each document into bullet points of key facts.
  2. Structure. Facts are packed into a tiny JSON sketch.
  3. Draft guidelines. The model writes Python @dataclass specs that define every label and field.
  4. Annotate. It fills those classes with spans from the text, giving clean training pairs.

A sanity script imports every file; anything that raises an error is dropped, removing hallucinated labels and spans.
